## Install

```sh
$ npm install --save-dev gulp-cordova-config
```

## Add the following to your gulpfile

```js
gulp.task('config', require('gulp-cordova-config'));
```
## Usage
```sh
$ gulp config --appId="com.new.id"
$ gulp config --appName="newApp"
```
